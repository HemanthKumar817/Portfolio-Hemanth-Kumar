# Portfolio Hemanth Kumar - Digital Portfolio

A highly interactive, immersive digital portfolio built with modern web technologies. This project showcases advanced UI/UX skills, featuring real-time fluid simulations, custom shaders, and dynamic theming.

## ‚ú® Features

-   **Immersive Background**: `LiquidEther` - a real-time, interactive WebGL fluid simulation powered by Three.js and custom shaders.
-   **Dynamic Theming**: Global color system controlled by a custom **Hue Wheel Color Picker**. All accents, glows, and borders adapt instantly.
-   **Advanced Animations**:
    -   **GSAP Target Cursor**: Magnetic, lag-free cursor tracking with dynamic hover states.
    -   **StarBorder**: Glowing, animated gradients for the Contact section.
    -   **Scroll-Triggered Reveals**: Smooth entry animations for project cards and skills.
-   **Responsive Design**: fully optimized for Desktop, Tablet, and Mobile devices with custom navigation.
-   **Modern Tech Stack**: Built with performance and scalability in mind.

## üì¶ Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/HemanthKumar817/Portfolio-Hemanth-Kumar.git
    cd Portfolio-Hemanth-Kumar
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Start Development Server**
    ```bash
    npm run dev
    ```

4.  **Build for Production**
    ```bash
    npm run build
    ```

5.  **Preview Production Build**
    ```bash
    npm run preview
    ```

## üöÄ Deployment

This portfolio is automatically deployed to GitHub Pages using GitHub Actions. Every push to the `main` branch triggers a new deployment.

### Manual Deployment

To deploy manually:

1. Ensure all changes are committed
2. Push to the `main` branch:
   ```bash
   git push origin main
   ```
3. The GitHub Actions workflow will automatically build and deploy the site

### GitHub Pages Setup

To enable GitHub Pages for your fork:

1. Go to your repository Settings
2. Navigate to Pages (under Code and automation)
3. Under "Build and deployment", select "GitHub Actions" as the source
4. The site will be available at: `https://[your-username].github.io/Portfolio-Hemanth-Kumar/`
